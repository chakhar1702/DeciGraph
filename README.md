# DeciGraph
==
DeciGraph is a novel decision graph model for visualising dominance based if-then decision rules. The construction of a decision graph  begins with a preprocessing phase to extract a binary decision matrix from the descriptions of decision rules. To prevent the creation of nonexistent rules, the generated binary decision matrix may require some transformation using either column duplication, column ordering, or a combination of both strategies. The transformed  binary decision matrix is then used to construct a directed acyclic decision graph, where nodes represent elementary conditions and decisions, and edges denote relationships between conditions and decisions. Each path from the root to a decision node in this graph corresponds to a decision rule. This paper introduces the fundamental concepts of decision graphs, alongside algorithms for their construction, linearisation, and exploitation. Both linearisation and exploitation algorithms are based on the Breadth-First Search graph traversal algorithm.
